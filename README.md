# m29

A Clojure library designed to shorten urls using the cryptographic url
shortener http://m29.us/

## Usage

`[com.thelastcitadel/m29 "0.1.0"]`

```clojure
(require 'com.thelastcitadel.m29)

(com.thelastcitadel.m29/shorten "http://www.google.com")
```
